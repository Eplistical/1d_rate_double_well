# 1d double well crossing rate, written in C++11

**Algorithms included**

- FP
- BFP
- CME
- BCME
- ModBCME
- IESH

- [cmake](https://cmake.org/)

- [HDF5](https://support.hdfgroup.org/HDF5/)
- [Boost](https://www.boost.org/)
- [misc](https://github.com/Eplistical/misc)
